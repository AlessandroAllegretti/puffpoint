# Screens : Puffpoint

## Welcome Screen
Wenn man die App startet komm ein ganz einfacher Screen mit der Aufforderung sich anzumelden oder zu Registrieren

## Login / Registrierungs-Screen
Ein Fenster mit den einfachen Eingaben wie Name, Geburtsdatum, Email, Passwort, ...

## Der Main-App-Screen
Einfache Karte mit von den Usern erstellten Punkten auf der Karte oben Links ein Knopf wo man auf die Einstellungen, Liste der Spots, Impressum, Datenschutz, FAQ und Über uns. Oben Rechts ist ein Icon wo man auf sein Profil gehen kann, da kann man alles sehen wie wann der account erstellt wurde, welche Spots von einem erstellt wurden, name, email, ... . Unten Rechts ist ein +-Icon mit dem kann man einen neuen Spot erstellen.
